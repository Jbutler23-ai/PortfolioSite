# Joseph Butler — Portfolio

Personal portfolio and resume site.

Built with React, Vite, Tailwind CSS, and framer-motion. The resume downloads as a PDF generated client-side using @react-pdf/renderer.

## Running locally

```bash
npm install
npm run dev
```

## Stack

- React + Vite
- Tailwind CSS v4
- framer-motion
- @react-pdf/renderer
